# API Automation Testing with REST Assured using Java

This repository contains API automation practice exercises using Java, Rest Assured, and TestNG.

## Tech Stack
- Java
- Maven
- Rest Assured
- TestNG

## Project Structure
- src/ : source code for API test cases
- suites/ : TestNG suite files
- pom.xml : Maven dependencies and build configuration

## Features Covered
- GET request
- POST request
- PUT request
- DELETE request
- Request specification
- Response validation
- TestNG execution

## Purpose
This project is used for practicing API automation concepts during course learning.
